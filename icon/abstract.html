This research aims to design and implement a reference frequenc y identification (RFID), keypad lock, and short message service (SMS) security system using an Arduino-based with alert system. The system is designed to provide a secure and efficient means of access control for using RFID. The proposed system utilizes RFID technology to authenticate users and keypad lock to enable access to the system. In addition, the system sends SMS alerts to the registered phone numbers in case of unauthorized access attempt. The performance of the system is evaluated through experimental testing of its different components, including RFID scanning efficiency, OTP generation, alarm triggering, LED indication, buzzer activation, SMS delivery, and bypass testing. The proposed system provides an effective means of securing various applications, including homes, offices, and laboratories.

Findings

1.The RFID scanning efficiency of the system was found to be high.
1.1.The RFID scanning process had a high success rate of accurately detecting and identifying registered RFID tags, indicating that the system's scanning efficiency was reliable and effective.
1.2.The scanning speed of the RFID system was tested and found to be fast and efficient, allowing for quick and easy access to the secured area.
2.The One time password (OTP) generation was found to be efficient, and the system was capable of generating a unique OTP for each access attempt.
2.1.The use of OTP enhances the security of the system by reducing the likelihood of unauthorized access. The unique OTP generated by the system ensure that only authorized personnel can gain access.
2.2.The efficient generation of OTP also improves the user experience by reducing wait times and delays in accessing secure areas. This can increase user satisfaction and adoption of the system.
3.The alarm triggering and LED indication system were found to be effective in alerting the user of an unauthorized access attempt.
4.The buzzer activation system was found to be effective in producing an audible alarm.
5.The SMS delivery system was found to be efficient in sending alerts to the registered phone numbers in case of an unauthorized access attempt.
5.1.The SMS delivery system was reliable, and alerts were received by the registered phone numbers in a timely manner, ensuring that any potential security breaches could be quickly identified and addressed.

Overall, the study showed that the proposed system is an effective means of securing various applications, including homes, offices, and laboratories. The system achieved high accuracy and efficiency in authenticating users and detecting unauthorized access attempts. The proposed system provides a comprehensive and efficient solution to the problem of unauthorized access attempts. The findings of this study contribute to the field of access control and security systems, providing a valuable solution to the problem of unauthorized access.
